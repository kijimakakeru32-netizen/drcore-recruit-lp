Dr.core 採用LP — 編集用ファイル一式（オフライン版）
=====================================================

【内容】
- drcore-lp.html … サイト本体（HTML/CSS/JavaScriptすべて1ファイルに内包）
- assets/        … サイトで使用する画像13点

【使い方（Codex等での編集）】
1. drcore-lp.html をエディタで開いて編集します。
   - 文章・デザイン(CSS)・動作(JavaScript)はすべてこの1ファイル内にあります。
2. ダブルクリックでブラウザでも開けます（assets/ と同じ階層に置いたまま開いてください）。
   画像は assets/ファイル名 の相対パスで読み込むため、インターネットがなくても表示されます。

【注意】
- drcore-lp.html と assets フォルダは必ず同じ階層に置いてください。
- og:image / twitter:image（SNSシェア用サムネイル）のみ、公開サイトの絶対URLを参照しています。
  これはSNS共有時のプレビュー画像用で、ローカル表示には影響しません。
- このファイルは公開中のManusサイトとは独立したコピーです。
  ここでの編集内容は公開サイトには自動反映されません。
